Requirements
e107 will only work with version 4.1 or newer of PHP. If you need to upgrade see http://php.net for details. If you are remote hosted contact your site administrator.

Installation Instructions
Fresh Install ---
  • If you wish e107 to use it's own database, create one using phpMyAdmin - you can call the database anything you wish.
  • Upload all the files to your server, leaving the file structure intact.
  • In your ftp client, change the file permissions of /config.php to 666 (CHMOD 666)
  • Open your browser and navigate to install.php (ie http://yourURL/install.php). Follow the instructions to install e107. If there are any problems you will be informed.
During the install process you will be asked to set up the main site admin, please write the details you enter down as for security purposes they cannot be retrieved later.

Upgrading your version/build ---
  • Backup your config.php file
  • Upload all files to your server EXCEPT config.php - don't overwrite this file
  • Navigate to upgrade.php in your browser and follow the instructions
For security reasons you should chmod your config.php file back to 644 after installation is complete.

You also need to chmod the two files (news.xml and news.txt) in the /backend directory to 666 for your xml generation to work properly.

Common problems
Most problems are related to not having a recent enough version of PHP installed on the server. If you keep returning to the same stage during install you are probably running a version of PHP older than 4.1.
You need cookies enabled in your browser to log in.
Your logo is situated at /themes/shared/logo.png, change this file to alter your logo
